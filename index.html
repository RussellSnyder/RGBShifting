DemoGithubRGB Shifting

This jquery plugin allows you to have RGB Shifting effect on your images (using canvas). checkout demo for more info about this effect.

How to use

First, download the RGB Shifting plugin from github and include it in your html page.

Now, you need to put a canvas element in your html where you want this image shown, use the following code for creating a canvas:
<canvas id='example'></canvas>

Let's call the plugin to put effect on image: 

$(document).ready(function(){
	$("#example").rgbshifting({
		image: 'image_url_here.jpg'
	});
});

Options

Here is the list of options:

Option Default Description
image * false url of image which you want to load or javascript Image object.
mode horizental horizental or vertical ?
radius 1.25% radius of rgb shifting layers, you can pass it in pixels or percent. if you want to pass it in pixels use integer value.
starred options are required.

Browser support

Browsers which supports canvas, supports this plugin. (Can I use canvas?)
